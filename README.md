# Nyaman Water Theme Park System

 Overview
This project is a Java Swing-based Water Theme Park Management System integrated with a MySQL database. It allows efficient management of customer bookings, ticket pricing, and records.

##Features
- Add customer records
- Calculate ticket pricing (Adult, Children, Senior)
- Apply discounts
- Search customer by name
- Display all customer records
- Update existing customer information
- Delete customer records

##Technologies Used
- Java (Swing GUI)
- MySQL Database
- JDBC

##How to Run
1. Open the project in NetBeans
2. Configure MySQL database:
   - Database name: `customerdb`
   - Table: `new_table`
3. Update database credentials in `CustomerDB.java`
4. Run `GUI.java`
